
# Medical Appointment Scheduling Agent (Production-Ready Backend)

This repository contains a fully functional, production-grade backend:
- LLM-powered scheduling assistant (GPT-4o-mini)
- Real tool-calling architecture
- Mock Calendly integration
- Full RAG system using ChromaDB
- SOLID folder structure
- Tests included
- Architecture diagram included

## Run locally:
pip install -r requirements.txt
uvicorn backend.main:app --reload
